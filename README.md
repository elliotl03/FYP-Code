# FYP-Code
Relevant code used in final year project

'corrected_data' applies the calibration file to raw spectra files and produces corrected spectra in a folder

'testTVIB_calculator' includes all code used to calculate values for vibrational temperatures in the project

'Te_Yacora' includes code and plots for model balmer ratios and calculations for electron temperature

'BALMER_RATIOS' calculates balmer ratios from corrected spectra files and creates a file to store all ratios

'PCA_test' includes all relevant PCA calculations and plots for PCA calculations used in the report

'fulcher_bands' and '230WattsTvib' are useful plotting codes used in the results section of the report
